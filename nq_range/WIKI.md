# NQ Range Trading — 详细设计文档

## 策略原理

区间交易假设：价格在一段时间内在支撑和阻力之间震荡，上下边界有一定的预测性。

## 区间识别（待研究）

### 候选方案 1：BB 带宽
- BB Width = (BB_upper - BB_lower) / BB_mid
- Width < 阈值 → 认为是区间
- 优点：计算简单；缺点：滞后

### 候选方案 2：CI + ADX
- CI > 61.8（高震荡）且 ADX < 20（无趋势）
- 两个条件同时满足才认为是区间
- 优点：两个独立指标确认；缺点：CI 与 MR 信号互斥的问题在这里不存在（区间交易入场条件不同）

### 候选方案 3：N 周期支撑阻力
- 近期 N 根 bar 高点聚集区 → 阻力
- 近期 N 根 bar 低点聚集区 → 支撑
- 在支撑区附近做多，在阻力区附近做空

## 入场逻辑（草案）

做多：价格接近支撑区 + CI > 61.8 + ADX < 20
做空：价格接近阻力区 + CI > 61.8 + ADX < 20

止盈：对面的边界
止损：突破边界一定幅度（ATR 倍数）

## 风险

最大风险：区间突破。若 ADX 快速上升或价格突破区间边界，需要快速止损。
