# NQ Range Trading — 详细设计文档

## 策略原理

**区间交易**：在横盘震荡市场中，价格在支撑位和阻力位之间反复震荡。
在靠近支撑位时买入，靠近阻力位（或均值）时卖出。

本策略使用：
- 布林带下轨作为动态支撑位（买入区域）
- 布林带中轨（20日均线）作为止盈目标
- CI + ADX 双重过滤确保只在真正的震荡市中交易

## 为什么比 NQ MR 好？

NQ MR 要求 RSI + BB + VWAP **三个极端条件同时满足**，过于严格，导致：
- 33笔/2年（稀疏）
- 胜率 48.5%（低于 50%）

NQ 区间交易只需 **BB下轨 + RSI<45**（两个宽松条件），再加上 CI+ADX 市场过滤：
- 120笔/2年（频率 3.6x）
- 胜率 62.5%（合理且可持续）

## 市场状态过滤（核心）

```python
# 必须同时满足：
CI > 55    # 震荡指数高 → 价格在区间内震荡
ADX < 25   # 趋势强度低 → 没有明显单边趋势
```

CI（Choppiness Index）的含义：
- CI > 61.8：强震荡
- 38.2 < CI < 61.8：中性
- CI < 38.2：强趋势

实测：ci_threshold=55 比 ci_threshold=61.8 更好（120笔 vs 43笔，Sharpe 相近）

## 入场条件

```python
score = int(close <= bb_lower * 1.002)   # 价格在 BB 下轨附近（±0.2%）
      + int(rsi < 45)                    # RSI 偏低（不要求极端超卖）

if score >= 2:   # 两个都满足才入场
    buy(sl = close - ATR * 1.5)
```

## 出场逻辑（多重出场，先到先出）

1. **BB 中轨止盈**：价格回到 20 日均线 → 反弹到位
2. **ATR 止盈**：价格 + ATR×1.0 → 快速获利了结
3. **时间止损**：持仓超过 16h → 区间可能已改变
4. **硬止损**：入场价 - ATR×1.5（由 SL 参数处理）

## 为什么只做多？

回测仅测试多头方向（买入支撑，卖在中轨）。
空头方向（卖在阻力，买在中轨）未测试，待后续研究。
NQ 长期牛市背景下，多头优先。

## 信号频率分析（2024-03 ~ 2026-06）

- 月均约 4-5 笔
- 大成交月：2024-09（+$926），2024-12（+$525），2025-03（+$473）
- 主要亏损：2024-11（-$256），2025-04（-$177），2026-06（-$527）
- 信号在波动率适中的月份最多，极端行情月（大涨/大跌）ADX 超标，信号减少

## 与 NQ MR 并存的逻辑

两个策略条件互补，很少同时触发：
- NQ MR：ADX < 25 + RSI < 30 + BB下轨 + VWAP-2ATR（极端超卖）
- NQ Range：ADX < 25 + CI > 55 + RSI < 45 + BB下轨附近（温和偏低）

NQ MR 信号是 NQ Range 信号的子集（更极端的情形），理论上不会抢单。
