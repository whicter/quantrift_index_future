# NQ Range Trading (nq_range)

NQ（MNQ）区间震荡交易策略。CI+ADX 识别横盘市场，在 BB 下轨附近买入，BB 中轨附近卖出。

## 状态：✅ 回测完成，待写引擎

## 策略概述

- **品种**：MNQ（$2/点）
- **周期**：1h，只做多
- **逻辑**：震荡市中在布林带下轨买入，中轨或 ATR 止盈出场
- **过滤**：CI > 55（震荡市确认）+ ADX < 25（无趋势确认）
- **与 MR 的区别**：MR 需要三合一极端信号；区间交易只需两个宽松条件

## 最优参数（2026-06-30 回测确认）

| 参数 | 值 | 说明 |
|------|---|------|
| ci_threshold | 55 | CI > 55 确认震荡市 |
| adx_threshold | 25 | ADX < 25 确认无趋势 |
| rsi_os | 45 | 比 MR 宽松（不要求极端超卖）|
| bb_mult | 2.0 | 布林带宽度 |
| sl_mult | 1.5 | 止损比 MR 宽（区间内允许更多波动）|
| tp_mult | 1.0 | 止盈在 ATR×1.0 或 BB 中轨（先到先出）|
| max_bars | 16 | 最长持仓 16h |
