# NQ Range Trading (nq_range)

NQ（MNQ）区间震荡交易策略。识别价格区间上下边界，在下边界买入、上边界卖出。

## 策略概述

- **品种**：MNQ（微型纳斯达克 100 期货，$2/点）
- **周期**：1h（可能多时框）
- **逻辑**：区间交易（双向，做多下边界 + 做空上边界）
- **过滤**：CI > 61.8（确认震荡市）+ ADX < 25
- **状态**：⏳ 待开发

## 与 NQ MR 的核心区别

| 对比项 | NQ MR | NQ 区间交易 |
|-------|-------|-----------|
| 入场条件 | 极端偏离（RSI+BB+VWAP 同时触发）| 到达区间边界 |
| 市场要求 | 允许短期急跌后反弹 | 需要明确横盘区间 |
| 方向 | 只做多（抄急跌底） | 双向（做区间两端）|
| 交易频率 | 低（约 1次/月） | 较高（区间内多次）|

## 区间识别方法（待研究）

候选方案：
1. **固定 BB 带宽**：BB 带宽 < N 时认为是区间
2. **支撑/阻力**：近期 N 根 bar 的高低点聚集区域
3. **CI + ADX 组合**：CI > 61.8 且 ADX < 20
