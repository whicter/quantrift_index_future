# NQ/ES Statistical Arbitrage (nq_es_spread)

NQ 与 ES 收益率价差策略。NQ 相对 ES 短期偏离均值时建仓，快速回归后平仓。

## 状态：✅ 研究完成，⚠️ 待模拟盘验证执行可行性

## 策略概述

- **品种**：MNQ（$2/点）+ MES（$5/点），1 MNQ + 2 MES（近似美元中性）
- **周期**：1h
- **信号**：1H 收益率价差 Z-score（滚动 240h 窗口）
- **入场**：|Z| > 2.0 → Z>0 空NQ多ES，Z<0 多NQ空ES
- **出场**：|Z| < 0.5（回归）或 |Z| > 3.5（止损）或 8h 超时

## 关键研究结论

### 协整检验结果（重要）
- Engle-Granger 协整检验：**p=0.255，❌ 不协整**
- ADF 平稳性检验（价格价差）：**p=0.102，❌ 不平稳**
- β（对冲比率）不稳定：范围 1.7 ~ 7.9，std=1.127

**含义**：NQ/ES 价格在统计上不协整，长期价差会漂移。但**短期（1-5h）高度相关**，短期偏离会快速修复。

### 收益率价差特性
- 半衰期：~0.8 bar（与时间级别无关，1H/2H/4H 一样）
- |Z|>2 频率：约 7-13%（机会充足）
- 1H 信号质量最好（机会最多，半衰期相同）

## ⚠️ 执行风险

回测使用 1H 收盘价，实盘必须注意：

1. **双腿同步执行**：MNQ + MES 需要几乎同时下单，任何一腿滑点都会影响价差
2. **半衰期仅 0.8 bar**：整点收盘时下单，下一个 bar 即可能已回归，入场时机极关键
3. **建议**：先纸交易观察 1 个月，确认实际执行价差与回测接近，再转实盘
